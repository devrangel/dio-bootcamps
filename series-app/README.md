### Projeto *"Criando um APP simples de cadastro de séries em .NET"* do bootcamp da DIO. Foi adicionado ao projeto um CRUD e utilização do Swagger para documentação e teste da API.

Foram utilizados:
* .NET 5
* InMemory Database
* Swagger
